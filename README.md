# Real-Time Image Classification System

## 📌 Overview

The Real-Time Image Classification System** is a Deep Learning application that performs real-time object recognition using a webcam. The project utilizes the pre-trained **MobileNetV2 Convolutional Neural Network (CNN)** model trained on the ImageNet dataset to classify objects captured by the camera and display predictions instantly.

This project demonstrates the practical implementation of Computer Vision and Deep Learning for real-time image classification.

---

## 🚀 Features

* Real-time webcam image capture
* Deep Learning based object classification
* Uses pre-trained MobileNetV2 model
* Displays prediction label with confidence score
* Fast and lightweight CNN architecture
* Easy to run and understand
* No additional training required

## 🎯 How It Works

1. MobileNetV2 pre-trained model is loaded.
2. Webcam captures live video frames.
3. Each frame is resized to 224×224 pixels.
4. Image preprocessing is performed.
5. The CNN predicts the object present in the frame.
6. The predicted class and confidence score are displayed on the screen.
7. Press **'q'** to exit the application.
